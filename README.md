Product Inventory System-
(Collections + Constructors + Keywords + Java 8).
Problem Statement-
Manage products in inventory.
Requirements-
Create Product class:
productId,  productName, price, quantity.
Use:
this, static, constructor chaining,
Store products in LinkedList<Product>.
Operations:
Add product,
Update product quantity,
Remove product,
Display out-of-stock products (Streams),
Calculate total inventory value.
Concepts Used:
Constructors & Keywords,
Collections,
Java 8 Streams,
Control flow.
